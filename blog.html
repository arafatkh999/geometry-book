What is DOM and what is the purpose of DOM?
Dom is actually Document Object Model.The purpose of dom is to make website interactive by using javascript.
How will you select HTML elements using DOM? Name the DOM methods.
We can select HTML elements using DOM in 4 ways.The DOM methods are given below:
1. getElementById
2. getElementByClassName
3. getElementByTagName
4. querySelector & querySelectorAll
What is event bubble?
Event bubble is a kind of event that can perform event listening to it's all items like to the top to bottom.
What is a callback function and why will you use it?
Callback function is a kind of function that can use into another funcation as an argument. When we want to use a function only after another function has completed it's execution then we can use a callback function.
